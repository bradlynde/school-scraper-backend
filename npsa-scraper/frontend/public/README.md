# Public Assets Directory

Place your logo image file here as `npsa-logo.png` (or update the filename in `components/Navigation.tsx` if using a different name/format).

Supported formats: PNG, JPG, SVG, WebP

The logo should be the exact NPSA logo with:
- "nonprofit" in dark blue (#1e3a5f)
- "security advisors" in olive-green (#6b8e23)
- Five green dots above the 'o' in "nonprofit"
